# Budget Automation

Este proyecto está diseñado para automatizar la gestión de presupuestos personales y empresariales.

## Características

- Automatización de registros de gastos
- Categorización automática de transacciones
- Generación de informes y análisis
- Seguimiento de presupuestos en tiempo real
